# RadioSphere

This application will provide three-dimensional geolocation, where you navigate through a representation of the terrestrial globe, listening to broadcasts from 8000 local radio stations around the globe.

Many thanks to Radio Garden (https://pt.wikipedia.org/wiki/Radio_Garden) which is founded and developed by German companies Studio Puckey and Studio Moniker in partnership with the Dutch Institute for Sound and Vision. Radio Garden acts as an interface for providing connection to 8000 radio stations around the globe.
